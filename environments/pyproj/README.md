# pyproj_pipeline_tools

various tools for manipulating/reprojecting EO data

## Installation

```
git clone git@github.com:surftemp/eo-pipelines.git
cd eo-pipelines/environments/pyproj
conda env create -f pyproj_env.yml
conda activate pyproj_env
pip install -e .
```

## Uninstall

```
pip uninstall pyproj_pipeline_tools
```
